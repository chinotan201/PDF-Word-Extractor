# PDF Word Extractor

A simple Python script to extract words from a PDF file using the `PyMuPDF` library (also known as `fitz`).

## Features

- Extracts all words from a PDF file.
- Words are returned as a list of strings, one per word.
- Can be easily modified for further text processing (e.g., word frequency analysis, cleaning, etc.).

## Requirements

To run this script, you'll need to install the following dependencies:

- `Python 3.x`
- `PyMuPDF` library

You can install the required dependencies with:

```bash
pip install pymupdf
